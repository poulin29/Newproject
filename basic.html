HTML Styles

There are six number of styles in HTML.They are:

  1. HTML style: The style property defines you to set the style of HTML elemnet.
  2. Background-color: The background colour property is used to define the Background color
    of the HTML element.
  3. HTML Color: The color property defines the color of the text of an HTML element.
  4. HTML font-family:The font family defines the style of the text of an HTML element.
  5. HTML font-size: This define the size of the text of an HTML document.
  6. HTML Text- Alignment: This defines the text to be aligned horizontally.
